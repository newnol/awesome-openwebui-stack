# Security

**Auth**, **secrets**, **sandboxing**, and **supply-chain** awareness when enabling tools and external APIs.

## Guidance for catalog contributors

- Call out tools that execute shell commands, run browsers, or fetch arbitrary URLs.
- Link to upstream security advisories when relevant.
- Prefer least-privilege API keys and scoped tokens in examples (never commit real secrets).
