# What is a “stack” in this repo?

A **stack** is a short **showcase** write-up under `stacks/<name>/README.md` for a **finished** setup you recommend. It must include an **Author** section with a **GitHub** profile or organization link. Example: [openwebui-stack](../stacks/openwebui-stack/README.md).

**Structure:** copy [template-full-stack/README.md](../stacks/template-full-stack/README.md), then trim sections that do not apply (e.g. omit Compose for a functions-only stack).

**Deploy files:** optional `docker-compose.yml` / `.env.example` beside `README.md`, or link an external infra repo.

List stacks in [catalog/stacks.md](../catalog/stacks.md); process in [submission guidelines](submission-guidelines.md).
