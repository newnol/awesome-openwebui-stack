# Stacks

**Showcase** recipes (finished setups worth copying). Not step-by-step “build Open WebUI from zero” tutorials—link your repos and official docs for that. Every stack `README.md` must include an **Author** section with a **GitHub** profile or organization URL.

| | Folder |
|---|--------|
| **Open WebUI stack** (Claude-style cloud, LiteLLM, Tavily, RAG outline) | [openwebui-stack/](openwebui-stack/) |
| **Template** (copy to add a new stack) | [template-full-stack/](template-full-stack/) |

Each stack is a directory with **`README.md`**. Add **`docker-compose.yml`**, **`.env.example`**, or other deploy files **next to** `README.md` when you want others to reproduce the setup.
